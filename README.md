
  # Maze Solver

  [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

  ## Description

  A depth first search maze solver. Generates a new maze and then uses dfs to solve. Built in Python.
